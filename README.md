# ChunChunCrimson LNReader Plugins

A custom LNReader plugin repository containing community-maintained sources for light novels and web novels.

Current focus:

* WorldEnd / SukaSuka extensions
* Fan translation sources
* Archive and mirror sources
* Related side stories and supplementary content

Additional plugins and sources may be added over time.

## Add to LNReader

Repository URL:

https://raw.githubusercontent.com/chunchuncrimson/my-lnreader-plugins/plugins/v3.0.0/.dist/plugins.min.json

## Notes

* Primarily created to support WorldEnd / SukaSuka readers.
* Additional novel sources may be added in future releases.
* Not an official LNReader repository.
* Plugin availability depends on the availability of the source websites.
* Some plugins may become unavailable if the corresponding website changes or goes offline.

## Disclaimer

This repository is an independently maintained fork of the LNReader plugin ecosystem and is intended to provide additional community-maintained sources.

The plugins and sources included here are selected and maintained by the repository owner and contributors. Inclusion of a source does not imply endorsement by LNReader, the original website operators, authors, publishers, or translators.

Content availability and functionality depend on the respective source websites and may change without notice.
